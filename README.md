# xinitrc-config

## Config Location
~/.xinitrc

## Start Xorg display server
```
$ startx
```

## Run different window manager or dekstop environment
```
$ startx ~/.xinitrc $Session
```
$Session is the name of the window manager/desktop environment.### 

### $Session options
Option 1) awesome

Option 2) i3

Option 3) fluxbox
